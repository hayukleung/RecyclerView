## NestedScrolling
> 嵌套滑动

> 为了支持嵌套滑动
> 子View必须实现NestedScrollingChild接口
> 父View必须实现NestedScrollingParent接口
> RecyclerView实现了NestedScrollingChild接口
> CoordinatorLayout实现了NestedScrollingParent接口
